# ReFrame

A web application that transforms scientific hypotheses into beautiful literary forms.

## Setup

Image generation uses [Pollinations.ai](https://enter.pollinations.ai). You need an API key:

1. Sign up at **https://enter.pollinations.ai** and get an API key (secret `sk_` or publishable `pk_`).
2. Create a `.env` in the project root with:
   ```bash
   VITE_POLLINATIONS_API_KEY=your_key_here
   ```
3. Restart the dev server after changing `.env`.

## Literary Forms

1. **Short Poem** 
2. **Soliloquy or Sonnet**
3. **Short Epic** 
4. **Song** 
5. **Fairytale, Fable or Myth** 
6. **Proverb**
